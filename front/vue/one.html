# vue-angle-picker

效果图

# 基本原理

三角函数：余弦函数、反三角函数求弧度、弧度转角度

# 使用

npm

npm install vue-angle-picker

Mount

import Vue from 'vue'
import anglePicker from 'vue-angle-picker'
vue.use(anglePicker)

demo:

<template>
 <angle-picker :angle='angle' @change="changeAngle"></angle-picker> 
</template>
 <script>
  export default{
   data(){
     angle:0
  },
  methods:{
   changeAngle(val){
     this.angle=val
    }
  }
}
</script>

# 配置项

  • angle(角度)
  • closeWise（defult:true,顺时针）
  • theme（风格：指针旋转模式，小球旋转模式）
