# 防抖（debounce）

# 什么是防抖

防抖：n秒内只执行一次，如果在n秒内再次触发，则重新计时。

# 节流（throttle）

# 什么是节流

节流：n秒内只执行一次，如果在n秒内再次触发，则忽略。
