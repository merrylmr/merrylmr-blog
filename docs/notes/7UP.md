# 人生七年

这是一个采访来自英国14个7岁小孩的纪录片，他们有的来自孤儿院有的是上层社会的小孩。从7岁开始，每隔7年拍摄一次，倾听他们的梦想，畅谈他们的生活。
在这一系列的纪录片中，可以看到他们的成长，他们的生活，他们的梦想，他们的家庭，他们的规划等
